# Calculadora Android

Este é um projeto de calculadora simples desenvolvido para Android utilizando **Jetpack Compose**. O aplicativo oferece funcionalidades básicas de cálculo com uma interface moderna e responsiva.

## 🚀 Funcionalidades

- Operações básicas: Soma, Subtração, Multiplicação e Divisão.
- Cálculo de porcentagem.
- Interface adaptativa (Suporte para modo Retrato e Paisagem/Telas grandes).
- Formatação numérica automática (padrão brasileiro com vírgula).
- Persistência de estado simples durante rotação de tela usando `rememberSaveable`.

## 🛠️ Tecnologias Utilizadas

- **Linguagem:** [Kotlin](https://kotlinlang.org/)
- **UI Framework:** [Jetpack Compose](https://developer.android.com/jetpack/compose)
- **Arquitetura:** Baseada em estados do Compose.
- **Build System:** Gradle (Kotlin DSL)
- **Design:** Material Design 3

## 📱 Requisitos

- **Android SDK:** Mínimo 24 (Android 7.0), Alvo 36.
- **Android Studio:** Ladybug ou superior recomendado.

## 📂 Estrutura do Projeto

- `app/src/main/java/br/com/jittecnologia/calculadora/MainActivity.kt`: Contém a lógica principal e a UI da calculadora.
- `app/src/main/java/br/com/jittecnologia/calculadora/ui/theme/`: Definições de cores, tipografia e tema (Material3).
